# Pyramid - Web Academy

Configure as variáveis de ambiente (.env) tanto no Frontend quanto Backend a partir dos arquivos .env.example respectivos. Após isso, para executar ambos os programas, rode:

Frontend

```
$ npm install
$ npm run dev
```

Backend

```
$ npm install
$ npm run setup:dev
```
